PicoFeed - PHP Library to manage Feeds
======================================

Features
--------

- No dependency (use only SimpleXML)
- Simple and fast
- Support of many formats: RSS 2.0, Atom (in progress)
- Feed writer and reader
- Import/Export OPML subscriptions
